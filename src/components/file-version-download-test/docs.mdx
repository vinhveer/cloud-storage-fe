## File version download test

Trang này dùng để test API **GET /api/files/\{id\}/versions/\{version_id\}/download** (tải xuống version cụ thể của file).

- Điền `File ID` và `Version ID`, sau đó bấm **Download version**.
- Các nút **Quick** giúp test nhanh các case: happy path (200), unauthenticated (401), version not found (404), forbidden (403).
