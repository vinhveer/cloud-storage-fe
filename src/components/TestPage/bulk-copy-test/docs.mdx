## Bulk copy test

Trang này dùng để test API bulk copy (`POST /api/bulk/bulk-copy`).

- Nhập `file_ids`, `folder_ids`, `destination_folder_id` rồi nhấn **Run bulk copy**.
- Hoặc dùng các nút **Quick** để test nhanh happy path và lỗi COPY_FAILED.

> Nhớ đăng nhập trước để có token hợp lệ.
