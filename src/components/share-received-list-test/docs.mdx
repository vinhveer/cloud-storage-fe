## Received share list test

Trang này dùng để test API **GET /api/shares/received** (danh sách tất cả shares mà user hiện tại được chia sẻ).

- Nhập `page` và `per_page` rồi bấm **Fetch received shares**.
- Hoặc dùng các nút **Quick** để test nhanh: basic list (200), page=1/per_page=1 (200), unauthenticated (401).

> Nhớ đăng nhập trước để test 200. Để test 401, mở tab chưa đăng nhập rồi truy cập lại trang samples này.
