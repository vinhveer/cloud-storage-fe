## File version upload test

Trang này dùng để test API **POST /api/files/\{id\}/versions** (upload version mới cho file hiện tại).

- Điền `File ID`, `Action`, `Notes` (tuỳ chọn), chọn file PDF, sau đó bấm **Upload version**.
- Có các nút **Quick** để test nhanh các case: happy path (201), thiếu file (422), forbidden (403), not found (404).
