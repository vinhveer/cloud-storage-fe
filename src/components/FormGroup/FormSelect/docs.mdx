# FormSelect

Select với label, tùy chọn placeholder, bắt buộc, lỗi và trợ giúp.

## Mục đích & trường hợp xử lý

- Lựa chọn một giá trị từ danh sách có sẵn.

## Props

| Tên       | Kiểu                                                     | Mặc định | Mô tả                                       |
| --------- | -------------------------------------------------------- | -------- | ------------------------------------------- |
| label     | `string`                                                 | -        | Nhãn phía trên select                       |
| required  | `boolean`                                                | `false`  | Bắt buộc; thêm `*` và thuộc tính `required` |
| error     | `string \| null`                                         | -        | Thông báo lỗi dưới trường                   |
| help      | `string \| null`                                         | -        | Gợi ý/hướng dẫn dưới trường                 |
| options   | `{ value: string; label: string; disabled?: boolean }[]` | `[]`     | Danh sách lựa chọn                          |
| className | `string`                                                 | -        | Bổ sung class Tailwind                      |
| ...rest   | `React.SelectHTMLAttributes<HTMLSelectElement>`          | -        | Thuộc tính `select` gốc                     |

## Cách dùng

```tsx
<FormSelect
  label="Quốc gia"
  options={[
    { value: "vn", label: "Việt Nam" },
    { value: "us", label: "United States" },
  ]}
/>
```

#### Trạng thái

```tsx
{(() => {
  const [value, setValue] = React.useState('')
  return (
    <>
      <FormSelect label="Controlled" value={value} onChange={(e) => setValue(e.target.value)} help="Ví dụ controlled" options={[{ value: '', label: 'Choose...' }, { value: '1', label: 'Một' }, { value: '2', label: 'Hai' }]} />
      <FormSelect label="Có lỗi" error="Vui lòng chọn một giá trị" options={[{ value: 'a', label: 'A' }, { value: 'b', label: 'B', disabled: true }]} />
    </>
  )
})()}
```

## Lưu ý

- Khi không `required`, sẽ có option đầu là "Choose an option...".
- Dùng `disabled` trong option để vô hiệu hóa từng lựa chọn.
