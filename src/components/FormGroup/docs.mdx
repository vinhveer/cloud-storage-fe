# FormGroup

Container nhóm trường form với nhãn, lỗi và trợ giúp.

## Mục đích & trường hợp xử lý

- Gom nhóm các input liên quan, hiển thị nhãn chung, lỗi hoặc help.

## Props

| Tên | Kiểu | Mặc định | Mô tả |
| --- | --- | --- | --- |
| label | `string` | - | Nhãn cho nhóm |
| error | `string \| null` | - | Thông báo lỗi của nhóm |
| help | `string \| null` | - | Gợi ý/hướng dẫn |
| className | `string` | - | Bổ sung class Tailwind |
| ...rest | `React.HTMLAttributes<HTMLDivElement>` | - | Thuộc tính `div` gốc |

## Cách dùng
```tsx
<FormGroup label="Thông tin cá nhân" help="Vui lòng điền đầy đủ">
  <input className="border rounded px-3 py-2 w-full" placeholder="Họ tên" />
  <input className="border rounded px-3 py-2 w-full" placeholder="Email" />
</FormGroup>
```

### Demos

```tsx
import FormGroup from '@/components/FormGroup/FormGroup'
```

#### Cơ bản với subcomponents

- Mô tả: Gom nhóm input với nhãn chung, dùng `FormInput`.

```tsx
<FormGroup label="Thông tin cá nhân">
  <FormInput placeholder="Họ tên" />
  <FormInput type="email" placeholder="Email" />
</FormGroup>
```

<FormGroup label="Thông tin cá nhân">
  <FormInput placeholder="Họ tên" />
  <FormInput type="email" placeholder="Email" />
</FormGroup>

#### Nâng cao: Select/Checkbox/Radio

```tsx
<FormGroup label="Tùy chọn">
  <FormSelect options={[{ value: 'a', label: 'A' }, { value: 'b', label: 'B' }]} />
  <FormCheckbox label="Đồng ý điều khoản" />
  <FormRadio name="r1" options={[{ value: '1', label: 'Một' }, { value: '2', label: 'Hai' }]} />
</FormGroup>
```
