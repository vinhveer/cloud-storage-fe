# FormCard

Bố cục card cho form, căn giữa trang, hỗ trợ form nội bộ hoặc nội dung thường.

## Mục đích & trường hợp xử lý

- Tạo nhanh giao diện form (login/register/forgot) với Card.
- Dùng `action`/`method` để render form HTML; nếu không có, hiển thị nội dung thường.

## Props

| Tên | Kiểu | Mặc định | Mô tả |
| --- | --- | --- | --- |
| title | `string` | - | Tiêu đề trên `Card` |
| subtitle | `string` | - | Mô tả ngắn dưới tiêu đề |
| action | `string` | - | URL gửi form; nếu không có, hiển thị nội dung thường |
| method | `'GET' \| 'POST' \| 'PUT' \| 'PATCH' \| 'DELETE'` | `'POST'` | Phương thức form; `GET/POST` dùng trực tiếp, còn lại dùng `_method` hidden |
| csrfToken | `string` | - | Token CSRF (ẩn) khi `method !== 'GET'` |
| formProps | `React.FormHTMLAttributes<HTMLFormElement>` | - | Thuộc tính bổ sung cho thẻ `form` |
| footer | `React.ReactNode` | - | Vùng footer dưới `Card` |
| cardClassName | `string` | - | Bổ sung class cho `Card` |
| className | `string` | - | Bổ sung class cho wrapper ngoài |
| ...rest | `React.HTMLAttributes<HTMLDivElement>` | - | Thuộc tính `div` gốc |

## Cách dùng
```tsx
<FormCard title="Đăng nhập" subtitle="Chào mừng trở lại" action="/login" method="POST">
  <div className="space-y-4">
    {/* Fields ở đây */}
  </div>
  <button className="btn btn-primary btn-md w-full">Đăng nhập</button>
  
</FormCard>
```

#### Nội dung thường (không phải form)

- Mô tả: Hiển thị nội dung thông tin, liên kết.

```tsx
<FormCard title="Welcome" subtitle="This is a centered card">
  <p>Nội dung thường không phải form, ví dụ hướng dẫn.</p>
  <a href="#" className="btn btn-secondary btn-sm">Explore</a>
</FormCard>
```

#### Với form HTML

- Mô tả: Render form khi có `action`/`method`; tự thêm hidden `_method`/`_token` khi cần.

```tsx
<FormCard title="Đăng nhập" subtitle="Chào mừng trở lại" action="/login" method="POST">
  <div className="space-y-4">
    <label className="block">
      <span className="text-sm font-medium text-gray-700 dark:text-gray-300">Email</span>
      <input type="email" className="mt-1 block w-full border rounded-md px-3 py-2" placeholder="you@example.com" />
    </label>
    <label className="block">
      <span className="text-sm font-medium text-gray-700 dark:text-gray-300">Mật khẩu</span>
      <input type="password" className="mt-1 block w-full border rounded-md px-3 py-2" />
    </label>
  </div>
  <button className="btn btn-primary btn-md w-full">Đăng nhập</button>
</FormCard>

#### Form đăng ký (kết hợp subcomponents)

- Mô tả: Sử dụng `FormInput`, `FormSelect`, `FormCheckbox` trong `FormCard`.

```tsx
{(() => {
  const [agree, setAgree] = React.useState(false)
  return (
    <FormCard title="Đăng ký" subtitle="Tạo tài khoản mới">
      <div className="space-y-4">
        <FormInput label="Họ tên" placeholder="Nguyễn Văn A" />
        <FormInput label="Email" type="email" placeholder="you@example.com" />
        <FormSelect label="Quốc gia" options={[{ value: 'vn', label: 'Việt Nam' }, { value: 'us', label: 'United States' }]} />
        <FormCheckbox label="Tôi đồng ý điều khoản" checked={agree} onChange={(e) => setAgree(e.target.checked)} />
      </div>
      <button className="btn btn-primary btn-md w-full mt-4" disabled={!agree}>Tạo tài khoản</button>
    </FormCard>
  )
})()}
```

## Lưu ý

- Khi `method` không phải `GET/POST`, thành phần tự thêm input ẩn `_method`.
- Khi có `csrfToken` và `method !== 'GET'`, tự thêm input ẩn `_token`.
- Dùng `cardClassName` để tùy biến `Card` bên trong.
