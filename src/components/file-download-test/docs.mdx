## File download test

Trang này dùng để test API tải file (`GET /api/files/{id}/download`).

- Nhập `File ID` rồi nhấn **Download file**.
- Hoặc dùng các nút **Quick** để test nhanh các case trong spec.
- Khi tải thành công, sẽ hiện link để bạn click và lưu file.

> Nhớ đăng nhập trước để có token hợp lệ.
