## File copy test

Trang này dùng để test API sao chép file (`POST /api/files/{id}/copy`).

- Nhập `File ID`, `Destination Folder ID`, tick `Only latest version` nếu cần, rồi nhấn **Copy file**.
- Hoặc dùng các nút **Quick** để test nhanh các case trong spec.

> Nhớ đăng nhập trước để có token hợp lệ.
