## Alert

Hiển thị thông báo theo trạng thái, hỗ trợ icon, tiêu đề, nội dung và nút đóng.

### Để làm gì

- Hiển thị trạng thái nhanh: success, error, warning, info.
- Cung cấp nút đóng để người dùng tự ẩn (`dismissible`).

### Khi nào sử dụng

- Khi cần thông báo ngắn gọn trong luồng thao tác.
- Khi cần nhấn mạnh trạng thái kết quả của một hành động.

### Demos

```tsx
import Alert from '@/components/Alert/Alert'
```

- Mô tả: Dùng để hiển thị thông báo ngắn gọn theo 4 trạng thái phổ biến.

```tsx

```

```tsx
<Alert type="success" heading="Success" message="Everything worked!" />
```


#### Nội dung tuỳ chỉnh

- Mô tả: Dùng khi cần nội dung dài/tuỳ chỉnh (VD: văn bản nhiều dòng, link...).

```tsx
<Alert type="info" heading="Heads up!">
  <p>We just updated our policy. Please review the changes.</p>
</Alert>
```

### API

| Prop | Type | Default | Mô tả |
| --- | --- | --- | --- |
| `type` | `'success' | 'error' | 'warning' | 'info'` | `'info'` | Kiểu thông báo |
| `heading` | `string \| null` | `null` | Tiêu đề (tuỳ chọn) |
| `message` | `string` | `''` | Nội dung văn bản ngắn |
| `dismissible` | `boolean` | `true` | Hiển thị nút đóng |
| `icon` | `React.ReactNode` | `—` | Ghi đè icon mặc định |
| `onDismiss` | `() => void` | `—` | Gọi khi đóng alert |
| `...rest` | `React.HTMLAttributes<HTMLDivElement>` | `—` | Thuộc tính HTML bổ sung |

### Ghi chú

- Icon mặc định dùng SVG nội bộ, có thể thay bằng `icon`.
- Màu sắc theo `type` sử dụng Tailwind.
