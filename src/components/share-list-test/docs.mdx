## Share list test

Trang này dùng để test API **GET /api/shares** (danh sách tất cả shares mà user hiện tại đã tạo).

- Nhập `page` và `per_page` rồi bấm **Fetch shares**.
- Hoặc dùng các nút **Quick** để test nhanh: basic list (200), page=2/per_page=1 (200), unauthenticated (401).

> Nhớ đăng nhập trước để test 200. Để test 401, mở tab chưa đăng nhập rồi truy cập lại trang samples này.
