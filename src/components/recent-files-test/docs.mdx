## Recent files test

Trang này dùng để test API danh sách file gần đây (`GET /api/files/recent`).

- Nhập `Limit` (tùy chọn) rồi nhấn **Fetch recent files**.
- Hoặc dùng các nút **Quick** để test nhanh default (20) và `limit=2`.

> Nhớ đăng nhập trước để có token hợp lệ.
